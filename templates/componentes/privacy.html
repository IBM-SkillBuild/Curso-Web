1. INTRODUCCIÓN

Bienvenido a {{nombre_del_sitio_web}}. Respetamos su privacidad y nos comprometemos a proteger sus datos personales. Esta política de privacidad le informará sobre cómo cuidamos sus datos personales cuando visita nuestro sitio web y le informará sobre sus derechos de privacidad.

2. DATOS QUE RECOPILAMOS

Podemos recopilar, usar, almacenar y transferir diferentes tipos de datos personales sobre usted, incluyendo:

  • Datos de identidad: Nombre, apellido, nombre de usuario o identificador similar
  • Datos de contacto: Dirección de correo electrónico, número de teléfono, dirección postal
  • Datos técnicos: Dirección IP, datos de inicio de sesión, tipo y versión del navegador
  • Datos de perfil: Su nombre de usuario y contraseña, sus preferencias
  • Datos de uso: Información sobre cómo utiliza nuestro sitio web y servicios

3. CÓMO UTILIZAMOS SUS DATOS

Utilizaremos sus datos personales solo cuando la ley nos lo permita. Más comúnmente, utilizaremos sus datos personales en las siguientes circunstancias:

  • Para registrar su cuenta
  • Para proporcionarle nuestros servicios
  • Para responder a sus consultas
  • Para mejorar nuestro sitio web y servicios
  • Para cumplir con obligaciones legales

4. DIVULGACIÓN DE SUS DATOS PERSONALES

Podemos compartir sus datos personales con:

  • Proveedores de servicios que utilizamos para apoyar nuestro negocio
  • Autoridades fiscales, reguladoras u otras autoridades cuando sea requerido por ley

5. SEGURIDAD DE DATOS

Hemos implementado medidas de seguridad apropiadas para evitar que sus datos personales se pierdan, usen o accedan de manera no autorizada.

6. SUS DERECHOS LEGALES

Bajo ciertas circunstancias, usted tiene derechos bajo las leyes de protección de datos en relación con sus datos personales, incluyendo el derecho a:

  • Solicitar acceso a sus datos personales
  • Solicitar la corrección de sus datos personales
  • Solicitar la eliminación de sus datos personales
  • Oponerse al procesamiento de sus datos personales
  • Solicitar la restricción del procesamiento de sus datos personales
  • Solicitar la transferencia de sus datos personales
  • Retirar el consentimiento
